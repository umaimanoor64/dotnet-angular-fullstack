# Dotnet Angular Fullstack Application

This repository contains a full-stack web application built using **ASP.NET Core (Web API)** for the backend and **Angular** for the frontend.

The project follows clean architecture principles with a modular and scalable structure, implementing RESTful APIs, secure authentication, and modern frontend development practices.

## 🚀 Tech Stack

### Backend
- ASP.NET Core Web API
- Entity Framework Core
- SQL Database
- JWT Authentication
- Clean Architecture Pattern

### Frontend
- Angular
- TypeScript
- Angular Services & Routing
- Reactive Forms
- HTTP Client Integration

## ✨ Features
- RESTful CRUD operations
- Secure authentication & authorization
- Modular folder structure
- Environment-based configuration
- API integration with Angular frontend

## 📂 Project Structure
- `/Backend` – ASP.NET Core Web API
- `/Frontend` – Angular Application

This project demonstrates full-stack development practices including API design, state management, frontend-backend integration, and scalable application structure.
